The main differnce between list and dict is the type of object it stores.
In list we can store differnt items like string, int and boolean all in one list and its mutable.
In dictionary we store in the form of key value pair. we have a value against each key similar to object we have in javascript.
In dictionaries the key must be unique

When to use each :
List : when we want to store things in order and we have many items of same type.
Dict: when we want to access something with unique identifier and with labels and when we want to get each thing with more detail

Example: 
If we want to store 5 number we can use list 
If we want to store profile we can use dict
If we want to store 10 profile we can use list of dictiories.
